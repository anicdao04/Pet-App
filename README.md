## About PetApp

PetApp is a web based application intend to meet both adopter and donator. It also has the features to report lost and found pets, email notification coming from the app to a future adoptor who sent a request for a specific pet using app form, and many more.
Laravel is accessible, powerful, and provides tools required for large, robust applications.
